# odin-tyr
A TUI interface in Odin

## Requirements

`libc`, uses [`termios`](https://www.man7.org/linux/man-pages/man3/termios.3.html).

## Naming
Because if you try to search for "odin tui",
most search engines think you misspelled Tyr, from Norse.
